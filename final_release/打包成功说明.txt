======================================
         用户使用说明
======================================

1. 系统要求:
   - Windows 7/8/10/11 (64位)
   - 2GB+ 内存
   - 网络连接

2. 首次使用:
   - 双击 YuYinTiHuan.exe 启动
   - 菜单 -> 设置 -> 配置API
   - 开始转换视频

3. API配置:
   - 科大讯飞STT API (语音转文字)
   - 科大讯飞TTS API (文字转语音)  
   - 百度翻译API (中英互译)
